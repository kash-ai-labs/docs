## حالت کیوں اہم چھ؟

کٲشُر منز معنی صرف لفظ ترتیب سان نَہ بنان؛ **حالت** تہٕ **پوسٹپوزیشن** معنی ہُنٛد بڑا حصہ سنبھالان۔

## بُنیادی حالتہٕ

یہِ سادہ عملی خلاصہ چھ:

| حالت | عام کام |
| --- | --- |
| فاعلی / سادہ | جُملہُک عام فاعل |
| دَتیو | کَسِس کُن، کَسِس خاطر، تجربہٕ کرنہٕ وٲل |
| اضافی | مِلکیت، تعلق |
| مقامی | جگہ، اندر، پٮ۪ٹھ، نَزدیک |
| عامل یا ایجنٹو نما | کینہہ ماضی بناوٹَن منز |

## پوسٹپوزیشن کِیاہ چھ؟

پوسٹپوزیشن سُہ جزو چھ یُس اسم یا ضمیر **پَتہٕ** آیہِ۔ یہِ پیش لفظ پٲٹھۍ پہلے نہٕ آیہِ بلکہ بعد منز آیہِ۔

### سادہ نمونہٕ

```text
اسم / ضمیر + حالت / بدلی شکل + پوسٹپوزیشن
```

## دَتیو ہُنٛد خاص مقام

کٲشُر منز دَتیو صرف "کو" مطلب نہٕ چھ۔ یہِ احساس، ضرورت، یاد، خبر، خواہش، یا اہلیت پَتھ بھی آسہِ۔

مثال طور:

- یَسِس کچھ یاد آسہِ
- یَسِس کینہہ معلوم چھ
- یَسِس کینہہ چاہیئے

اِنہٕ جملَن منز دَتیو سوچنہٕ ہُنٛد طریقہٕ بدلِتھ چھ۔

## اضافت

مِلکیت یا تعلق ظاہر کرنہٕ خٲطرٕ اضافی بناوٹ استعمال گژھِہ۔ بعض جگہہٕ یِم بناوٹ اسمُک جنس یا عدد سۭتھ جُڑِتھ نظر آیہِ۔

## مقام ہُنٛد تعلق

کٲشُر منز "منز، پٮ۪ٹھ، نِش، کُن" قسم ہٕند رشتہ پَرنہٕ بہت ضروری چھ۔ یِم سَرٕف لغت نہٕ بلکہ جُملہ ہُنٛد صحیح مطلب صاف کران۔

## سیکھنہٕ ہُنٛد طریقہ

ہَر جُملہ پَر یہِ سوال کَرو:

1. فاعل کون چھ؟
2. کَسِس کُن کَم وٕچھان؟
3. مقام چھ؟
4. مِلکیت چھ؟
5. پوسٹپوزیشن کُس لفظ پَتہٕ آیہِ؟

## خلاصہ

کٲشُر گرائمر ہُنٛد مضبوط سمجھ یِمنہِ وٲرتہٕ بدون نہٕ بنان:

- حالت بدلِنہٕ ہُنٛد سُراغ
- دَتیو ہُنٛد استعمال
- اضافت
- مقام تہٕ نسبت
- پوسٹپوزیشن

## آن لاین قاعدہٕ نوٹ

- کٲشُر منز بنیادی حالت: نامزد، دَتیو، ایرگیٹو، اَبلیٹو، ندا شمار کرنہٕ آمن۔
- حالت اکثر noun لاحقہ سۭتھ ظاہر گژھان۔
- مقامی، آلہ نما، اضافی، ساتھی نما، طرفی معنی اکثر پوسٹپوزیشن سۭتھ ظاہر گژھان۔
- سادہ ماضی منز مفعولی فعل سۭتھ ایرگیٹو بناوٹ آیہِ؛ باقی حالتن منز نامزد-دَتیو نمونہٕ عام چھ۔
